# Strong-password-checker
🔐 Password Strength Checker (Python)

This Python program checks whether a password is strong or weak based on common security rules.

🚀 Features
Validates password length (minimum 8 characters)
Checks for uppercase and lowercase letters
Ensures at least one numeric digit
Requires at least one special character
Gives clear feedback on why a password is weak
🛠️ How It Works

The program takes a password as input and validates it using conditions:

❌ Less than 8 characters → Weak
❌ Missing uppercase letter → Weak
❌ Missing lowercase letter → Weak
❌ Missing digit → Weak
❌ Missing special character → Weak
✅ If all conditions pass → Strong Password
